# 코인 플리퍼 프로

재미있는 동전 던지기 게임으로 운을 시험해보세요!

## 기능

- 🪙 실시간 동전 던지기 애니메이션
- 📊 결과 통계 추적
- 🎨 아름다운 UI/UX
- 📱 반응형 디자인
- 🚀 빠른 로딩 속도

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 기술 스택

- React 19
- TypeScript
- Tailwind CSS
- Vite
- React Router

## 배포

이 프로젝트는 Netlify, Vercel, GitHub Pages 등에서 쉽게 배포할 수 있습니다.

### Netlify 배포
1. GitHub에 코드 업로드
2. Netlify에서 저장소 연결
3. 빌드 명령어: `npm run build`
4. 배포 폴더: `out`
